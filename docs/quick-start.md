# Quick Start Guide

Get productive with ThinkPod in 5 minutes.

## Your First Note

1. **Create a new thought**: Press `Cmd+N` (Mac) or `Ctrl+N` (Windows/Linux)
2. **Write something**: Type your idea, thought, or note
3. **Save**: Press `Cmd+S` or just close the window (auto-saves)

That's it! Your note is now in your Vault as a markdown file.

---

## Talk to Wilfred

1. **Open the agent chat**: Press `Cmd+J`
2. **Ask a question**: Type "Summarize this note"
3. **Press Enter**: Wilfred will read your current note and respond

Wilfred automatically knows what document you're viewing — no need to copy/paste.

---

## Organize Your Notes

### The Sidebar

Click the folder icon (or press `Cmd+B`) to see your file tree.

Your Vault has these categories:
- **Journal** - Daily reflections, logs
- **Ideas** - Brainstorms, concepts
- **Projects** - Work in progress
- **People** - Notes about individuals
- **Others** - Everything else

Drag and drop notes to organize them.

### Drafts

Unfinished thoughts go in **Drafts**. When ready, move them to the appropriate category.

---

## Voice Capture

1. **Click the microphone icon** in the toolbar
2. **Speak** your thought
3. **Click stop** when done

Wilfred will transcribe your speech into text. Edit as needed, then save.

---

## Essential Workflows

### Quick Capture

The core ThinkPod workflow:
1. Idea strikes → `Cmd+N`
2. Write it down → `Cmd+S`
3. Keep going

Don't worry about organization yet. Capture first, organize later.

### Expand an Idea

1. Write a rough draft
2. `Cmd+J` to open Wilfred
3. Ask: "Expand this with examples"
4. Review Wilfred's suggestions
5. Incorporate what works

### Research a Topic

1. Create a new note about the topic
2. `Cmd+J` to open Wilfred
3. Ask: "Research [topic] and summarize key points"
4. Wilfred will search the web and provide insights
5. Edit and save

### Daily Journaling

1. `Cmd+N` to create a new note
2. Title it with today's date (e.g., "2024-03-15")
3. Write your thoughts
4. Save to **Journal** folder

## Tips for Beginners

### Start Simple

Don't overthink organization. Just write. Wilfred can help categorize later.

### Use Markdown

ThinkPod uses markdown formatting:
- `# Heading` for titles
- `**bold**` for emphasis
- `- item` for lists
- `[link](url)` for links

But plain text works fine too!

### Ask Wilfred Anything

Wilfred can:
- Summarize long notes
- Suggest improvements
- Answer questions about your writing
- Research topics
- Connect ideas across notes

Don't be shy — experiment!

### Keyboard > Mouse

Learn the essential shortcuts:
- `Cmd+N` - New note
- `Cmd+J` - Wilfred
- `Cmd+B` - Sidebar
- `Cmd+S` - Save

See [Keyboard Shortcuts](./shortcuts.md) for the full list.

## Common Beginner Questions

### "Where are my notes stored?"

In your Vault folder (the one you chose during setup). They're plain markdown files you can open with any text editor.

### "Can I use ThinkPod offline?"

Yes! If you're using a local AI model (Ollama, LM Studio), everything works offline. Only web search requires internet.

### "What if I don't like Wilfred's suggestion?"

Ignore it! Wilfred is a collaborator, not a dictator. Use what helps, discard the rest.

### "How do I delete a note?"

Right-click the note in the sidebar and select Delete. Or delete the markdown file directly from your Vault folder.

### "Can I use ThinkPod with my existing notes?"

Yes! Point your Vault to a folder with existing markdown files. ThinkPod will read them.
