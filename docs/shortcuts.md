# Keyboard Shortcuts

ThinkPod is designed to be keyboard-first. Learn these shortcuts to work faster.

> **Note**: On macOS, the primary modifier is `⌘` (Command). On Windows/Linux, it's `Ctrl`.

---

## Navigation

| Shortcut | Description |
|----------|-------------|
| `Cmd+N` / `Ctrl+N` | Create new thought |
| `Cmd+B` / `Ctrl+B` | Toggle sidebar |
| `Cmd+J` / `Ctrl+J` | Toggle Agent Chat (Wilfred) |
| `Esc` | Close panels / Go back |

---

## Document Editing

| Shortcut | Description |
|----------|-------------|
| `Cmd+S` / `Ctrl+S` | Save current document |
| `Cmd+Enter` / `Ctrl+Enter` | Save and close (in New Thought) |
| `Cmd+Enter` / `Ctrl+Enter` | Send message (in Inbox/Chat) |

---

## Agent Chat

| Shortcut | Description |
|----------|-------------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `↑` | Edit previous message (when input empty) |

---

## File Tree

| Shortcut | Description |
|----------|-------------|
| `F2` | Rename selected file/folder |
| `Enter` | Confirm rename |
| `Esc` | Cancel rename |

---

## Tips

### Muscle Memory

The most important shortcuts to memorize:
1. `Cmd+N` - Quick capture is the core of ThinkPod
2. `Cmd+J` - Wilfred is your thinking partner
3. `Cmd+S` - Save your work frequently

### Workflow Example

A typical flow using only keyboard:
1. `Cmd+N` - Create new thought
2. Type your idea
3. `Cmd+S` - Save
4. `Cmd+J` - Ask Wilfred to expand it
5. `Esc` - Close chat
6. Continue writing

### Customization

Keyboard shortcuts can be customized in **Settings → Shortcuts** (coming soon).

---

## Platform Differences

### macOS
- Primary modifier: `⌘` (Command)
- Secondary modifier: `⌥` (Option)

### Windows/Linux
- Primary modifier: `Ctrl`
- Secondary modifier: `Alt`

Most shortcuts use the primary modifier, so they work consistently across platforms.
