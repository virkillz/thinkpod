# What is the strategy to make this application special?

## Agent know user favorite topic. 
How? 

We have mechanism to add tags into notes and we can rank the tag based on occurence. This provide strong foundation. 

However, it is possible the tags is not accurately reflect user interest. During conversation with user (session log) or during read the user notes, agent might find new topic of interest. Imagine in the chat session user said "i really like oranges" it should be noted.

## Agent Knowledge base
Agent build knowledge base about user. In the form of folder with progressive disclosure. 

Agent once in a while will send few things to the user inbox. the goal for user is to feel:
- informed
- inspired
- entertained.


Cognitive scheduler end product should be skills improvement.
